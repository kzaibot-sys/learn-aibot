# PRD: AiBot — Аудит, исправления и современный редизайн

**Дата:** 2026-03-27
**Версия:** 1.0
**Статус:** Утверждён

---

## 1. Введение / Обзор

AiBot — LMS-платформа для онлайн-курсов (целевой рынок: Казахстан, русскоязычная аудитория). Предыдущая итерация разработки оставила критические баги, незавершённые фичи и устаревший дизайн. Данный PRD описывает полный цикл исправлений, доработок и редизайна платформы.

**Ключевой принцип:** Платформа для УЖЕ оплативших клиентов — БЕЗ платёжного UI.

## 2. Цели

1. Исправить все критические баги (невидимый landing, сломанное мобильное меню, неправильный подсчёт уроков)
2. Полностью убрать следы "LearnHub" и payment UI
3. Построить работающую систему геймификации (XP, уровни, ачивки, стрик, лидерборд)
4. Модернизировать дизайн Landing и LMS до современных стандартов (2026)
5. Улучшить архитектуру кода (сервисный слой, валидация, типизация)
6. Обеспечить 100% прохождение build и QA-чеклиста

## 3. Пользовательские истории

### Студент
- **US-1:** Как студент, я хочу видеть красивый landing с описанием курсов, чтобы понять что предлагает платформа
- **US-2:** Как студент, я хочу видеть реальное количество уроков в курсе, чтобы оценить объём обучения
- **US-3:** Как студент, я хочу получать XP за прохождение уроков и видеть свой уровень
- **US-4:** Как студент, я хочу разблокировать ачивки за достижения (первый урок, стрик 7 дней и т.д.)
- **US-5:** Как студент, я хочу видеть лидерборд и сравнивать свой прогресс с другими
- **US-6:** Как студент, я хочу получить сертификат по завершении курса (100%)
- **US-7:** Как студент, я хочу пользоваться платформой с телефона (responsive)
- **US-8:** Как студент, я хочу переключать тему (светлая/тёмная) и язык (RU/KZ)

### Администратор
- **US-9:** Как админ, я хочу видеть аналитику по геймификации (XP, ачивки)
- **US-10:** Как админ, я хочу управлять доступом студентов к курсам

## 4. Функциональные требования

### FR-1: Исправление багов
1. Все секции landing видимы и анимированы через CSS (не framer-motion)
2. Мобильное бургер-меню открывается/закрывается корректно
3. VIDEO уроки показывают видеоплеер, TEXT уроки — текстовый контент
4. Форма логина очищается после logout
5. Каталог курсов показывает реальное количество уроков

### FR-2: Брендинг
6. Вся платформа использует бренд "AiBot" (нигде нет "LearnHub")
7. Tagline: "AI-Powered Learning Platform" / "Образовательная платформа с ИИ"
8. Favicon: SVG с роботом

### FR-3: Удаление payment UI
9. Нет страницы pricing/billing/subscription в навигации
10. Нет premium бейджей и кнопок покупки на карточках курсов
11. Админ-панель управления платежами сохранена (для просмотра истории)

### FR-4: Система геймификации
12. При завершении урока студент получает 25 XP + бонус за стрик (15 XP)
13. Система уровней: каждый следующий уровень требует на 20% больше XP
14. 10 ачивок: First Step, Gaining Momentum, Scholar, Graduate, Multidisciplinary, Consistent, On a Roll, Iron Will, Speed Reader, Thousander
15. Стрик: подсчёт дней подряд с активностью
16. Лидерборд: топ-50 пользователей по XP
17. При завершении урока — XP popup + achievement toast (при разблокировке)

### FR-5: Сертификаты
18. Сертификат доступен при 100% прохождении курса
19. Уникальный номер формата AIBOT-XXXXX-XXXX
20. Страница сертификатов с градиентными карточками

### FR-6: Современный дизайн
21. Landing: hero с gradient mesh, ScrollReveal анимации, glassmorphism карточки
22. LMS: современные course cards, progress bars, stats виджеты
23. Тёмная тема: продуманная палитра (Slate 900 background, Slate 800 cards)
24. Mobile-first: работает от 320px

## 5. Не-цели (вне рамок)

- Платёжная система (payments disabled, admin grants access)
- Telegram Mini App редизайн
- Bot AI рефакторинг
- Регистрация новых пользователей (login only)
- Мультиязычный контент курсов (только UI i18n)

## 6. Дизайнерские решения

### Палитра
- Primary gradient: Indigo (#6366f1) → Violet (#8b5cf6)
- Accent: Cyan (#06b6d4)
- Dark background: Slate 900 (#0f172a)

### Принципы
- Mobile-first responsive (Tailwind breakpoints)
- CSS-only анимации (NO framer-motion на landing)
- Glassmorphism карточки (backdrop-blur, полупрозрачные borders)
- Micro-interactions на hover (scale, translateY)
- Skeleton loading для каждой страницы

### Компоненты
- shadcn/ui: Card, Button, Badge, Progress, Accordion, Dialog
- Custom: XPBar, AchievementCard, LeaderboardRow, CertificateCard, ScrollReveal

## 7. Технические соображения

### Database
- Добавить в User: totalXp, level, streak, lastActiveAt
- Новая модель: UserAchievement
- Добавить Certificate → Course relation
- Индекс на users.total_xp для leaderboard

### Backend (apps/api)
- Сервисный слой: `services/gamification.ts`
- Zod валидация на всех endpoints
- Rate limiting на gamification endpoints

### Frontend (apps/web)
- React Query hooks для API
- IntersectionObserver для scroll анимаций
- Loading/Error boundaries на каждом route
